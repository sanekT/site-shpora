Хитрости таблицы

Рамки(border)

Допустим есть таблица, и если прописать у неё атрибут border="1", то таблица отобразится с такими рамками:

Какой то текст Какой то текст
Какой то текст Какой то текст
Какой то текст Какой то текст
Какой то текст Какой то текст
Какой то текст Какой то текст
Какой то текст Какой то текст

Без этого атрибута таблица будет полностью без рамок:

Какой то текст Какой то текст
Какой то текст Какой то текст
Какой то текст Какой то текст
Какой то текст Какой то текст
Какой то текст Какой то текст
Какой то текст Какой то текст

Если же таблице задать в стилях рамку таблицы вот так:

          table{
              border: 1px solid black;
          }

То рамка будет только у самой таблицы:

Какой то текст Какой то текст
Какой то текст Какой то текст
Какой то текст Какой то текст
Какой то текст Какой то текст
Какой то текст Какой то текст
Какой то текст Какой то текст

Но нам надо добиться рамки как при атрибуте border="1" в первом варианте, но уже прописать в стилях. Можно применить такое свойство:

          table{
              border-width: 1px;
          }

Вроде дожна была рамка отобразиться,однако рамка не отображается, если не установлен отличный от нуля атрибут border или стиль border-style. Добавим тогда ещё свойство border-style:

          table{
             border-style: solid;
             border-width: 1px;
          }

Тогда вроде всё работае как при первом варианте при атрибуте border.
