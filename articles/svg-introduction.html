Знакомство с SVG

SVG — это формат векторной графики. В отличие от растровой графики — PNG, GIF, JPEG — SVG может растягиваться и сжиматься без потери качества. Также SVG-элементы можно оформить с помощью CSS и добавить им интерактивности с помощью JavaScrip. Вот простой пример кода:

            <svg style="width: 400px; height: 200px; background-color: white;">
              <circle r="50" cx="50%" cy="50%" fill="yellowgreen"/>
            </svg>

SVG-элемент вставляется с помощью тега svg, внутри которого уже находится остальное содержимое: фигуры, картинки или текст. SVG можно встраивать несколькими разными способами, мы рассмотрим их позже, а сейчас будем вставлять его непосредственно в код страницы.

Рисуем прямоугольник

В SVG есть несколько способов нарисовать фигуру, для простых фигур есть свои теги. Например, прямоугольник рисуется с помощью тега rect. Код простого прямоугольника выглядит так:

            <svg style="width: 400px; height: 200px; background-color: white;">
                <rect width="150" height="100" fill="orange"/>
            </svg>

Обратите внимание: все теги в SVG должны быть закрыты, то есть должно быть так: <rect .../> или так: <rect...></rect>. Мы будем использовать первый способ.

Атрибуты width и height управляют, соответственно, шириной и высотой фигуры. Значения можно задавать и в пикселях, и в процентах. Для значений в пикселях после значения не нужно писать px, потому что пиксели — единица измерения, используемая в SVG по умолчанию. Проценты рассчитываются относительно размеров всего SVG-изображения: горизонтальные значения относительно ширины, вертикальные — относительно высоты. В современных браузерах размерами и положением фигур нельзя управлять через CSS, но эта возможность появится в будущем.

Координаты прямоугольника

Чтобы задать координаты прямоугольника, используются атрибуты x и y:

            <svg style="width: 400px; height: 200px; background-color: white;">
                <rect width="50%" height="100" fill="orange" x="20%" y="50" />
            </svg>

Координаты определяют положение верхнего левого угла фигуры.

Скругление углов

Скруглением углов прямоугольника управляют параметры rx и ry. Атрибут rx задаёт скругление по горизонтали, а ry — по вертикали. Если атрибут ry не задан, он будет равен rx.

            <svg style="width: 400px; height: 200px; background-color: white;">
                <rect width="50%" height="100" fill="orange" x="20%" y="50" rx="50" ry="20" />
            </svg>

Многоугольники

В SVG можно рисовать не только четырёхугольники, но и многоугольники, это делается с помощью тега polygon. Вершины отмечаются так: "x,y x,y x,y" Пример кода:

            <svg style="width: 400px; height: 200px; background-color: white;">
                <polygon points="70,5 90,41 136,48 103,80 111,126 70,105 29,126 36,80 5,48 48,41"/>
            </svg>

В атрибуте points задаются координаты вершин фигуры. Каждая координата задаётся по x и y. Координаты в points нельзя задавать в процентах.

            <svg style="width: 400px; height: 200px; background-color: white;">
                <polygon points="5,135 115,5 225,135" fill="violet"/>
            </svg>

Рисунок

Рисуем окружность

Окружность рисуется с помощью тега circle. Пример кода:

            <svg style="width: 400px; height: 200px; background-color: white;">
                <circle r="50" fill="blue"/>
            </svg>

Атрибут r это радиус окружности. В отличии от других фигур, положение окружности определяется координатами цетра фигуры: атрибут cx задаёт положение по горизонтальной оси, cy - по вертикальной. По умолчанию координаты центра окружности равны 0,0, поэтому она находится в левом верхнем углу. Подвигаем фигуру:

            <svg style="width: 400px; height: 200px; background-color: white;">
                <circle r="50" cx="100" cy="50%" fill="blue"/>
            </svg>

Рисуем эллипс

Эллипс рисуется почти также как и круг, но у него 2 радиуса: по горизонтальной оси - rx и по вертикальной - ry.

            <svg style="width: 400px; height: 200px; background-color: white;">
                <ellipse rx="30" ry="40%" fill="crimson"/>
            </svg>

Расположение эллипса так же как и для circle задаётся с помощью cx и cy.

            <svg style="width: 400px; height: 200px; background-color: white;">
                <ellipse rx="30" ry="40%" cx="50%" cy="50%" fill="crimson"/>
            </svg>

Рисуем линии

Линии рисуются с помощью тега line. Координаты начала линии задаются атрибутами x1 и y1, координаты конца - атрибутами x2 и y2. Координаты можно задавать в процентах.

Так как линия не образует фигуру с внутреним контуром, для отображения ей нужно задать не заливку, а обводку. Обводкой управляют два атрибута: stroke(цвет) и stroke-width(толщина).

            <svg style="width: 400px; height: 200px; background-color: white;">
                <line x1="220" y1="20" x2="20" y2="90" stroke="violet" stroke-width="5"/>
            </svg>

Можно задать только цвет, тогда толщина будет равна 1px.

Рисуем ломаные линии

Ломаные линии рисуются с помощью тега polyline. Координаты точек на линии задаются в атрибуте points, как и для polygon.

            <svg style="width: 400px; height: 200px; background-color: white;">
                <polyline points="10,135 100,10 55,135 10,10" fill="none" stroke="crimson"/>
            </svg>

Разница между polygon и polyline заключается в поведении обводки: у многоугольника обводка сама по себе замыкается, а у ломаной линии остаётся незамкнутой.

            <svg style="width: 400px; height: 200px; background-color: white;">
                <polyline points="50,180 100,20 150,180 20,80 180,80 50,180 100,20" fill="none" stroke="crimson"/>
            </svg>
