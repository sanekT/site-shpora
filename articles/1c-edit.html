Полезные заметки

Сочетание клавишь по встроенному редактору(конфигуратору)

  1. Ctrl + F - посик по модулю
  2. Ctrl = H - замена
  3. Закладки поставить/снять Alt + F2 - если много кода в модуле, то ставим закладки на интересующей нас строке и слева появится серый квадратик, затем можно переключаться между этими закладками по нажатию F2.
