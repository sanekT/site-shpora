Полезные заметки

Сочетание клавишь по встроенному редактору(конфигуратору)

  1. Ctrl + F - поиск по модулю
  2. Ctrl + H - замена
  3. Закладки поставить/снять Alt + F2 - если много кода в модуле, то ставим закладки на интересующей нас строке и слева появится серый квадратик, затем можно переключаться между этими закладками по нажатию F2.
  4. Ctrl + F1 - поставив курсор на какой нибудь встроенный метод и нажав эту комбинацию, откроется синтакс-помошник сразу с описанием данного метода
  5. Ctrl + Пробел - дописать слово. Что то типо Emmet.
  6. Alt + Shift + F - наведеие красоты выделенного кода.
  7. Ctrl + стрелка вверх/вниз - открывает и закрывает папки с помощью клавиатуры. Гораздо быстрей это делается чем мышкой.
