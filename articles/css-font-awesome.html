Font-Awesome

Первый вариант

Заходим на этот сайт и жмём кнопку скачать. Скачали, находим на компьюторе, распаковываем и копируем себе в проект папку font и из папки css копируем файл font-awesome.css в свою папку css в проекте. И подключим их в head на страничке

< link rel="stylesheet" href="css/font-awesome.css">

Затем на сайте переходим в меню на страничку "Иконки", листаем вниз и ищем интересующую иконку. Когда нашли кликаем на неё и заходим на страничку с разными размерами этой иконки. Там под иконкой копируем юникод.

Если мы хотим вставить картинку в инпут то обернём наш инпут в div и ему пропишем псевдеэлемент before:

    div::before{
        font-family: "FontAwesome";
        content: '\f007';//это наш скопированный юникод
        position: absolute;
        font-size: 30px;
        padding: 10px 0 0 7px;
        color: green
    }

Это примерные стили, иконка в инпуте будет выглядеть примерно так:

К самому инпуту нельзя добавить псевдоэлементы, потому что у инпута нет контента. Прочитал в инете, сам толком пока не понимаю что это значит.

Если же нам надо вставить эти шрифтовые иконки просто где то в потоке, то на сайте также переходим на страничку с иконкой и внизу показан тег, который мы копируем и втавляем в html

< i class="fa fa-vk" aria-hidden="true">< /i>

Вставляем сколько нужно нам иконок и для удобной работы с этими шрифтами обернём их в блок div например с классом social.
