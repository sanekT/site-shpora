PT Mono

Это добавляем в стили(обычно в начале css файла)

            @font-face {
                font-family: 'PT Mono';
                src: url('../fonts/PT Mono.ttf') format('truetype');
                font-weight: normal;
                font-style: normal;
                font-display: block;
            }

затем создаём папку fonts и в неё копируем этот шрифт. Он лежит по этому адресу - D:\Web\JS\Шрифты\Классные шрифты\PT Mono. Выглядет так:
