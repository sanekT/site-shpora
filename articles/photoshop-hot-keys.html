Горячие клавиши и разные особенности фотошопа

Горячие клавиши

  • Ctrl + Shift + C - скопировать в буфер то, что выделили инструментом "Прямоугольная область"
  • Ctrl + N - создаём новое окно куда можно вставить что то из буфера.
  • Ctrl + R - скрывает показывает линейку, правой кнопкой по линейки выбрать ед. измерения.
  • Ctrl + H - скрывает/показывает сетку
  • Ctrl + J или схватив мышкой слой перетащить его вниз на значок создания нового слоя - делаем копию слоя
  • Ctrl + G - выделенный слой помещается в созданную группу
  • Ctrl + Shift + U - обесцветить(сделать чёрно-белой) изображение на выбранном слое.
  • Ctrl + E - выделяем 2 слоя и при нажатии этих клавиш они объединятся в один слой.

Разные фишки

Чтобы вырезать картинку для веб, нужно выделить слой, для этого выбираем инструмент "Перемещение", самый верхний слева и надо в панели этого инструмента поставить галочку "Автовыбор" и левой кнопкой кликнуть по картинке в макете, справа выделиться нужный слой с картинкой. Затем првой кнопкой по этому слою и выбрать в меню Экспортировать слой.

Чтобы нанести на текст градиент, то выделаем слой с текстом и жмём правую кнопку мыши, в меню выбираем Параметры наложения - Нанесение градиента.
