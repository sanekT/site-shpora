for(let key in obj)

Файл articles/js-cicles.html

for(let key in obj){} - перебирает как свои, так и унаследованные свойства объекта, кроме свойства Symbol(). В key попадает ключ объекта. Чтобы отсеять унаследованные свойства можно воспользоваться методом obj.hasOwnProperty(key)

    for(let key in obj){
        if(obj.hasOwnProperty(key)){
            //если зашли сюда, то key родной ключ obj
        }
    }

Также можно воспользоваться методом Object.keys(obj). Он также вернёт массив родных ключей obj кроме унаследованных и свойств типа Symbol()

for(let key of arr)

for(let key of arr){} - цикл для перебора массивов и строк.

Различия в циклах

  1. Если в массиве есть пустые значения равные undefined, то при переборе его в обычном for(), эти значения undefined тоже выведутся, в for...in они не выведутся, в for...of выведутся.
  2. Отличие for...in от for...of в том, что в переменную key попадает ключ, а в for...of туда попадает сразу значение(for...of используется тогда, когда не нужны ключи)
