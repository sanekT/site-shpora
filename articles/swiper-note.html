Заметки о swiper

Подключение

Чтобы подключить и начать работать со слайдером нужно на этой странице посмотреть это делается.

Опции

direction: 'vertical' - слайдер перематывается вертикально, по умолчанию горизонтально.

slidesPerView: 1 - отображает кол-во слайдов. По умолчанию 1, также есть значение auto. С 00:27:45 рассказывает про разную ширину слайдов и значение auto у этой опции.

spaceBetween: 0 - задаёт margin-right для слайда.

slidesPerGroup: 1 - задаёт сколько будут слайдов прокручиваться за один клик. По умолчанию прокручивается по одному слайду

Чтобы запретить перелистывать слайды с помощью мыши, нужно к контейнеру слайдера с классом .swiper-container добавить ещё один класс .swiper-no-swiping

slidetoCklickedSlide: true - кликая по слайду можно перелистывать до него, не работает если стоит класс swiper-no-swiping

centeredSlides: true - активным теперь становится центральный слайд вместо левого по умолчанию.

Пагинация

В видео выше с 00:08:05 коротенько рассказывает как обращаться и стилизовать точечки внизу слайдера

clickable: true - по умолчанию при клике по пагинации мы не можем переключать слайды, а с этой опцией можно. Добавляетс в раздел пагинации.

Чтобы вынести пагинацию ниже слайдов смотрим видео с 00:21:00. Если коротко то надо просто swiper-container задать padding-bottom например 30 или 40px

С 00:22:35 рассказывает как сделать цифры вместо точек в пагинации. Надо просто в секции пагинации в фалйе js поставить опции type значение fraction.

Как увеличить центральный слайд

С 00:29:48. В своей недавней вёрстке мне нужен был такой функционал, и я посмотрев видео просто добавил такие клаcсы слайдеру

            .swiper-slide{
                transition: 0.5s;
            }
            .swiper-slide-active{
                transition: 0.5s;
                transform: scale(1.16);
                z-index: 1;
                box-shadow: 3px 3px 14px $baseColor;
            }

А дальше просто подгоняем всякие мелочи.
