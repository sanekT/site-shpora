Подключение Vue.js

Подключение прям в странице index.html

В отличии от Angular и React, Vue легко подключить прям в странице index.html. Для этого как обычно внизу после тега body в теге script атрибута src прописываем удалённый путь до Vue, ну или скачиваем его в файл и подключаем этот файл. Есть версия для разработки и продакш версия. Подключаем для разработки пока и 2ю версию(3ю пока не трогаю) - <script src="https://cdn.jsdelivr.net/npm/vue@2/dist/vue.js"></script>. Далее нужно чтобы в html был тег div с id="app":

            <body>
	
                <div id="app">
                    <input type="text" v-model="name">
                    <hr>
                    <h2>Привет, {{ name }}</h2>
                </div>
                
        
                <script src="https://cdn.jsdelivr.net/npm/vue@2/dist/vue.js"></script>
                <script>
                    new Vue({
                        el: '#app',
                        data: {
                            name: ''
                        }
                    });
                </script>
            </body>

Вот небольшой кусок кода, который при вводе имени в инпут тут же его выводит в шаблон {{ }}. Так подключать Vue на страницу не надо, потому что сделать такое может и jQuery и плюс для SEO это не очень из-за наличия таких атрибутов как v-model или шаблона {{ }}. Используется ли Vue хоть где нибудь с вот таким подключением пока не знаю. Вообще такое подключение задумывалось для того, чтобы попробовать что из себя представляет Vue.

Подключение через Vue CLI

Чтобы установить Vue CLI надо иметь скачаный node js и npm. Далее чтобы установить его глобально пишем команду:

npm install -g @vue/cli

Проверяем установился ли пакет командой:

vue --version или vue -V

Помощь для Vue команд vue -h или vue --help

Ещё один способ проверки установился ли модуль - это команда:

npm ls -g --depth=0 - эта команда покажет все модули самого верхнего уровня, т.е. установленных глобально.

При вызове команды vue появится список команд, которые мы можем использовать.

Чтобы создать новый проект с помощью vue cli мы набираем команду:

vue create <имя проекта> - Проект будет создан в ту папку где мы сейчас находимся.

Либо можно создат проект через графический интерфейс, для этого нужно вызвать команду:

vue ui

После того как проект создался и подкачались все пакеты из списка package.json можно посмотреть все скачанные пакеты командой

npm ls --depth=0

Далее начинаем работать с нашими файлами, но чтобы увидеть результат нужно перейти в наш проект и запустить команду:

nmp run serve

Нам напишут, что нужно в браузере открыть url http://localhost:8080
