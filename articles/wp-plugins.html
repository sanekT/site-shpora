Различные полезные плагины

Плагины Lorem. Lorem Shortcode

Этот плагин позволяет заполнять наши записи текстом-рыбой. Также этот плагин может вставлять шаблон картинки:

           [lorem]  -  вставит текст из 5 абзацев, в каждом абзаце по 3 предложения
           [loremimage]  -  вставит картинку по умолчанию 300х200

Подробнее, что можно сделать с помощью этого плагина, можно посмотреть нажав на Детали у самого плагина.
