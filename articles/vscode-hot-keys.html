Горячие клавиши

  • Alt + Левая кнопка - размножаем курсор
  • Ctrl + Alt + Стрелки вверх или вниз - также размножают курсор по линии.
  • Alt + Стрелки вверх или вниз - переносит строку.
  • Shift + Alt + Стрелки вверх или вниз - дублирует строку.
  • Shift + Alt + F - выравнивает код.
  • Ctrl + K, затем Ctrl + 0 - сворачивает все блоки и стили
  • Ctrl + G - открывается вверху панелька, куда вводим число и редактор переносит на строку с кодом под этим номером
  • Ctrl + D или зажатый Alt - ставим курсор на нужный тег и сочетанием Ctrl + D последовательно выбираем все такие одинаковые теги. Ну или зажимает Alt и левой кнопкой мыши двойным щелчком выбираем нужные нам теги.

Расширения которыми пользуюсь

  • Sass
  • Material Theme
  • Vetur
  • Scss Formater
  • Auto Close Tag
  • Auto Rename Tag
  • Bracket Pair Coloriser
  • CSS Peek
  • Path Intellisense
  • eCSStractor for VSCode - позволяет выделить html и контекстном меню выбрать копировать что то там связанное с cstractor и затем вставить в файле css, вставятся селекторы из html кода
  • LiveServer - автоматически обновляет браузер при изменении кода. После установки этого плагина внизу в строке можно увидеть кнопку "Go Live", по нажатию на которую запуститься сервер.
  • Wrap Console Log() - по горячим клавишам раскрывает словосочетание console.log() в разных случаях.
  • Live Sass Compiler - плагин, который позволяет писать на препроцессоре sass и тут же переводить на css. Установили и запускаем снизу кнопочкой Watch Sass и далее запускаеv Live Server и пишем на препроцессоре и он автоматом будет обновлять страницу и переводить в чистый css. Только нужно создать папку css и в ней 2 файла с одинаковым названием - один с раширением .scss, другой с расширением .css. Всё, пишем в файлике .scss.
