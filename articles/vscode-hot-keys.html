Горячие клавиши

  • Alt + Левая кнопка - размножаем курсор
  • Ctrl + Alt + Стрелки вверх или вниз - также размножают курсор по линии.
  • Alt + Стрелки вверх или вниз - переносит строку.
  • Shift + Alt + Стрелки вверх или вниз - дублирует строку.
  • Shift + Alt + F - выравнивает код.
  • Ctrl + K; Ctrl + 0 - сворачивает все блоки и стили

Расширения которыми пользуюсь

  • Sass
  • Material Theme
  • Vetur
  • Scss Formater
  • Auto Close Tag
  • Auto Rename Tag
  • Bracket Pair Coloriser
  • CSS Peek
  • Path Intellisense
  • eCSStractor for VSCode - позволяет выделить html и контекстном меню выбрать копировать что то там связанное с cstractor и затем вставить в файле css, вставятся селекторы из html кода
  • LiveServer - автоматически обновляет браузер при изменении кода. После установки этого плагина внизу в строке можно увидеть кнопку "Go Live", по нажатию на которую запуститься сервер.
