Meta информация

< meta charset="UTF-8" > - Устанавливаем кодировку страницы

< title >Заголовок< /title > - Document title in browser tabs

< link rel="shortcut icon" href="icon.ico" type="image/x-icon" > - Document Favicon in browser tabs(задаём картинку рядом с заголовком в закладке страницы)

< link rel="apple-touch-icon" href="apple-touch-icon.ico" >
< link rel="apple-touch-icon" sizes="72x72" href="apple-touch-icon-72x72.png" >
< link rel="apple-touch-icon" sizes="114x114" href="apple-touch-icon-114x114.png" > - Additional approach to add favicon(продвинутый способ задания иконки)

< meta name="description" content="Test web page" > - Краткое описание (или аннотация) страницы часто используется поисковиками при отображении результатов поиска.

< meta name="keywords" content="html, webdev, test" >
< meta name="author" content="Name author" > - SEO

< meta name="viewport" content="width=device-width, initial-scale=1" > - For mobile devices

< meta name="viewport" content="width=device-width, initial-scale=1 user-scalable=no" > - For mobile и no resize(запрещает зумить страницу)

< meta http-equiv="X-UA-Compatible" content="IE=edge" > - Use latest version of IE

< link rel="stylesheet" href="styles.css" > - For external styles(подключение внешней таблицы стилей)

< link rel="stylesheet" href="styles.css" > - Local styles fonts(подключение шрифтов через стили)

< link href="https://fonts.googleapis.com/css?family=Roboto&display=swap" rel="stylesheet" > - Google fonts(подключение шрифтов напрямую от гуглов)
