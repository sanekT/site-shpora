Функции битрикса

В основном шаблоне

  • $APPLICATION->ShowTitle() - выводит то что задано в системе через $APPLICATION->SetTitle("О компании");, если в системе задать $APPLICATION->SetPageProperty('TITLE', 'Заголовок о компании');, то $APPLICATION->ShowTitle() уже будет выводить то что задано через SetPageProperty(), но если $APPLICATION->ShowTitle(false), то он начнёт снова выводить то что задано в $APPLICATION->SetTitle("О компании");. То есть как объяснили, то через SetPageProperty() задаётся для поисковиков и будет отображаться в заглавии закладки браузера, а для h1 и др. заголовков задаётся в $APPLICATION->SetTitle("О компании");, только когда выводить буду в h1, то нужно указать в скобках false - $APPLICATION->ShowTitle(false). По крайней мере так объяснил Salavey.
  • $APPLICATION->ShowHead() - подключает для вывода в шаблоне сайта основные поля тега: мета-теги Content-type, robots, keywords, description; стили css; скрипты стандартные
  • $APPLICATION->ShowPanel() - выводит панель управления администратора. Нужно писать сразу после открывающегося тега body и в div с id = "panel": 
                    <div id="panel"><?php $APPLICATION->ShowPanel(); ?></div>
  • echo SITE_TEMPLATE_PATH - содержит путь к шаблону сайта, например - /local/templates/sparrow
  • echo SITE_DIR - содержит путь корня сайта - /
  • $APPLICATION->GetCurDir() - вернёт название папки, в которой находимся, например - /about/, на главной возвращает /. Если поставить аргумент true, то к папке ещё добавляет index.php, вот так - /about/index.php
  • $APPLICATION->GetCurPage() - пока заметил, что возвращает то же что и метод $APPLICATION->GetCurDir()
  • $APPLICATION->GetCurPageParam() - возвращает путь к текущей странице относительно корня c добавленными новыми и(или) удаленными текущими параметрами. Смотреть в API битрикса
  • CJSCore::Init(array("jquery")); - подключение jquery из ядра битрикса
  • echo $USER->GetEmail() - вернёт почту админа, если он авторизован
  • echo $USER->GetFormattedName() - вернёт имя админа
