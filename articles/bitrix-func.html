Функции битрикса

В основном шаблоне

  • $APPLICATION->ShowTitle() - выводит тайтл сайта, сама строка с названием тайтла вводится в админке. 
                    <title> <?php $APPLICATION->ShowTitle(); ?> </title>
  • $APPLICATION->ShowHead() - подключает для вывода в шаблоне сайта основные поля тега: мета-теги Content-type, robots, keywords, description; стили css; скрипты стандартные
  • $APPLICATION->ShowPanel() - выводит панель управления администратора. Нужно писать сразу после открывающегося тега body и в div с id = "panel": 
                    <div id="panel"><?php $APPLICATION->ShowPanel(); ?></div>
  • echo SITE_TEMPLATE_PATH - подставляет путь к шаблону сайта, например - /local/templates/sparrow
  • $APPLICATION->ShowTitle(false) - заголовок(в h1 например использовать)
  • $APPLICATION->GetCurDir() - вернёт путь до текущей папки(директории)
