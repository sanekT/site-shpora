Vue CLI

Чтобы установить Vue CLI надо иметь скачаный node js и npm. Далее чтобы установить его глобально пишем команду:

npm install -g @vue/cli

Проверяем установился ли пакет командой:

vue --version

Ещё один способ проверки установился ли модуль - это команда:

npm ls -g --depth=0 - эта команда покажет модули самого верхнего уровня без вложенных элементов.

При вызове команды vue появится саписок команд, которые мы можем использовать.

Чтобы создать новый проект с помощью vue cli мы набираем команду:

vue create <имя проекта> - Проект будет создан в ту папку где мы сейчас находимся.

Либо можно создат проект через графический интерфейс, для этого нужно вызвать команду:

vue ui

После того как проект создался и подкачались все пакеты из списка package.json можно посмотреть все скачанные пакеты командой

npm ls --depth=0

Далее начинаем работать с нашими файлами, но чтобы увидеть результат нужно перейти в наш проект - cd <имя&rt; и запустить команду:

nmp run serve

Нам напишут, что нужно в браузере открыть url http://localhost:8080
