Полезные заметки

Сочетание клавишь по встроенному редактору(конфигуратору)

  1. Ctrl + F - поиск по модулю
  2. Ctrl + H - замена
  3. Ctrl + G - перейти к номеру строки в модуле.
  4. Закладки поставить/снять Alt + F2 - если много кода в модуле, то ставим закладки на интересующей нас строке и слева появится серый квадратик, затем можно переключаться между этими закладками по нажатию F2.
  5. Ctrl + F1 - поставив курсор на какой нибудь встроенный метод и нажав эту комбинацию, откроется синтакс-помошник сразу с описанием данного метода
  6. Ctrl + Пробел - дописать слово. Что то типо Emmet.
  7. Alt + Shift + F - наведеие красоты выделенного кода.
  8. Ctrl + F1 - открывает справку встроенного языка.
  9. Alt + F1 - открывает поиск по общей справке.
  10. Ctrl + стрелка вверх/вниз - открывает и закрывает папки с помощью клавиатуры. Гораздо быстрей это делается чем мышкой.
  11. Поставить курсор на функцию или процедуру и нажать F12 - перенесёт в функцию. Заем нажать Ctrl -+, чтобы вернуться обратно на вызов функции.
  12. Ctrl + Shift + Пробел - если поставить курсор после ( в какой нибудь функции или процедуре, то появится подсказка с параметрами этой функции.
  13. Ctrl + Insert, Shift + insert - выделяем строку кода нажав слева на серую полосу и копируем в буфер первым сочетанием и сразу жмём второе сочетание, чтобы вставить строку кода столько сколько нужно.
  14. Ctrl + Shift + "+", Ctrl + Shift + "-" - сворачивает-разворачивает области и остальные участки кода.
  15. Shift + Esc - закрыть текущее окно
  16. F12 - поставим курсор на название процедуры или функции и по нажатию перекинет в то место где описана эта процедура
  17. Ctrl + "-" - обратно вернёмся в том место, откуда мы провалились по нажатию F12
  18. Ctrl + T - если мы находимся в каком то модуле, то по нажатию нам откроется в дереве конфигурации тот объект, в модуле которого мы находимся
