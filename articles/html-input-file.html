Об input с типом file

Такой инпут это поле для ввода имени файла, который пересылается на сервер. Была задача показать слова "Прикрепить файл", по клику на которые нужно открыть окно для добавления файла и последующего его отправки на сервер. Но нам не нужно отображать стандартный вид этого инпута, поэтому мы в css скрываем этот инпут а вместо него выводим эти слова. Чтобы при клике по этим словам открывалось окно для выбора файла, мы обернём эти слова в лейбл и этот лейбл через атрибут for привяжем к нашему скрытому инпуту. Далее нам надо при выборе файла отобразить вместо этих слов имя выбранного файла. По умолчанию когда мы выбираем файл, то рядом с кнопкой "Выбрать файл" есть строка "Файл не выбран", но после выбора файла, туда записывается имя выбранного файла. Нам нужно как то получить эту строку с именем. Для этого в js получим наш скрытый инпут и у него есть свойство files, который возвращает псевдомассив FileList с выбранными файлами. Этот псевдомассив в индексах содержит объекты с данными о каждом загруженном файле. Например мы добавили один файл, идём ищем этот инпут и обращаемся к его свойству - let arr = input.files. Теперь в массиве лежит один объект под индексом 0 и у этого объекта есть свойство name, где и лежит имя нашего файла, получаем его и вставляем куда нам надо. Если нужна ещё какая инфа об файле, то можно посмотреть в этом объекте, что он ещё там содержит. Ещё добавлю, чтобы получить имя файла, нам надо на скрытый инпут повесить событие change или input потому как файл загрузится произойдёт одно из этих событий и обработчике этих событий мы получаем имя файла. Пока по этому инпуту всё.
