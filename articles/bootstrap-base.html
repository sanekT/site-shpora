Скачивание

Можно подключить через cdn, а можно скачать bootstrap. Пойдём по 2му варианту.

Чтобы начать работу с библиотекой перейдём на её сайт тут. Можно нажать "Начать работу" и перейти к компонентам или можно загрузить фреймворк нажав "Скачать", но при клике по "Скачать" переходитк к скачиванию 5й версии, вот ссылка на скачиванием 4й версии тут.

На выбор есть скомпилированный вариант css и js(это когда Sass файлы обработаны в css, добавлены преффиксы и минимизированы, как и js) и исходные файлы, для которых нужен компилятор Sass и постпроцессор, а также инструменты сборки для минимизации кода.

Качаю исходные файлы. Чтобы всё работало также нужно скачать jQuery тут. Создаю папку с проектом и из исходников bootstrap из папки dist копирую или вырезаю в мой проект 2 папки - css и js, также в папку js добавим скачаный jQuery. Теперь создаём index.html и можно работать.

Начинаем работу с bootstrap

Перейдём теперь на сайте слева в меню раздел "Начало работы", чуть прокрутим вниз и скачаем Начальный шаблон страницы в наш index.html. В области head в шаблоне прописана cdn ссылка link на сжатую версию bootstrap.min.css, но у нас уже скачаны исходники, поэтому стираем старый путь и прописывае туда такой путь - <link rel="stylesheet" href="css/bootstrap.css">, указывая основной файл bootstrap.css из нашей папки css.

В шаблоне также снизу видим, что прописан путь cdn для подключения jQuery, стираем этот путь и подключаем скачанный jQuery из папки js. Строку ниже с подключением bootstrap.bundle.min.js пока можно удалить. Адаптивная сетка

class="container"
Первое во что можно обернуть контент это класс container. Прописав <div class="container"></div> и вставив во внутрь него другие теги с содержимым, то бутстрап выровняет по центру этот контейнер и задаст ему ширину 1140px.

class="container-fluid"
Также есть класс container-fluid, он делает всё то же самое что и первый, но только расширяет контент на всю ширину экрана.

class="row"
Если сравнивать с таблицей, то это аналог тега tr, т.е. этот клас нам делаем строковый блок, в который мы помещаем какой то контент. Этот класс нужен для того, чтобы внутри него контент делить на колонки.
