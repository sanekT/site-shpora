Оформление SVG-фигур

Заливка

SVG-фигуры имеют богатые возможности оформления: им, как и HTML-элементам, можно задавать заливку цветом, градиентом или картинкой, но помимо этого также можно управлять отдельно прозрачностью заливки и обводки, а в качестве заливки можно использовать текст.

Также интересные возможности имеет обводка. Например, можно управлять видом обводки и сделать её точками, пунктиром или морзянкой, а ещё обводку можно сделать не только цветом, но также градиентом или картинкой.

Если заливка не задана, то по умолчанию заливка имеет чёрный цвет. Цвет заливки задаётся тегом fill либо аналогичным CSS свойством. Цвет заливки задаётся в формает HEX: fill="#ffaa45", либо в формате HSLA: fill="hsla(50, 100%, 50%, 0,5)", либо именнованным цветом: fill="orange"

Прозрачность

Прозрачностью заливки можно управлять ка с помощью тега fill-opacity, так и с помощью аналогичного CSS свойства. Значение задаётся числом от 0 до 1. Прозрачность работает для всех видов заливок, в том числе и для градиентов и паттернов.

Иногда бывает нужно полностью убрать заливку, например когда нужен только контур. Это можно сделать значением none у атрибута fill

Обводка

Обводки задаются с помощью нескольких атрибутов, причём цвет и толщина обводки задаются отдельно. Цвет задаётся атрибутом stroke, а толщина stroke-width, по умолчанию толщина 1 px. У этих атрибутов есть аналоги в CSS. Прозрачность обводки задаётся атрибутом stroke-opacity со значением от 0 до 1

Пример:

            <svg style="width: 400px; height: 200px; background-color: white;">
                <circle r="50" cx="50%" cy="50%" fill="none" stroke="orange" stroke-width="2"/>
            </svg> 
            //или в css
            sircle{
                stroke: orange;
                fill: none;
                stroke-width: 2;
            }

            <svg style="width: 400px; height: 200px; background-color: white;">

            </svg>
