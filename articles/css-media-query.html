Медиа запросы

Медиа запросы нужно писать внизу после всех стилей.

        body {
            background-color: red;
        }
        /* для планшетов и фаблетов */
        @media (max-width:768px) {
            body {
                background-color: green;
            }
        }
        /* для декстопов */
        @media (min-width:769px) {
            body {
                background-color: blue;
            }
        }

@import

Директива @import это подключение одного css файла в другой. Подключается так:

@import url(media.css);

Эта директива не может быть использованна в середине или в конце файла css, только в начале, но такие препроцессоры как sass могут её ставить где угодно. Вот например у меня не получилось подключить медиа запросы из другого файла css, потому что медиа запросы нужно писать в конце файла, но в конце нельзя писать директиву @import, мы подключим в начале файл с медиа запросами через эту директиву и всё равно не заработает, файл то подключится, но медиа запросы в начале не будут работать. Но в sass это всё можно делать.

Если всё же надо вынести медиа запросы в отдельный файл и сделать это всё без препроцессоров, то можно сделать так:

  • Пишем такую строку в теге head - < link rel="stylesheet" media="(max-device-width:480px)" href="css/media.css">
  • Затем в основном файле css пишем - @import url(media.css) (max-device-width:480px);
  • Ну и в самом файле media.css пишем уже стиль без директив @media, а просто стиль, который применится. когда экран станет меньше 480 пикселей. Можно выбирать и диапозон размеров, в котором включатся стили. Об этом можно почитать тут
