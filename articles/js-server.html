Это примерный файл сервер, на который можно отправялять запросы, например через fetch из JS

Файл будет отдавать POST данные пришедшие из формы

            <?php
                //чтобы с сервера послать ошибку, в файле php можно послать такой заголвок
                header("HTTP/1.1 404 Not Found");
                //или ответитть возвратом POST данных
                echo var_dump($_POST);
                //также можно задержать ответ  по типу setTimeout()
                sleep(4);
                
                //итоговый скрипт
                
                //header("HTTP/1.1 404 Not Found");
                sleep(4);
                echo var_dump($_POST);
            ?>
