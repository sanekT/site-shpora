Настройки Sublime Text3

Как подключить плагины в ST3. Нажимаем Сtrl + Shift + P, откроется окно и в нём пишем install package control, откроется окно управления пакетами и по названию пакета ищем его и устанавливаем.

Плагины для Bootstrap4

  • Bootstrap4 Autocomplite - автомотически подставляет классы из bootsrap4
  • Bootstrap4 Snippets - позволяет вставлять готовые участки кода. После его установки снова открываем окно Сtrl + Shift + P и пишем слово start, первым в списке должен высветиться этот спипет, жмём Enter и сразу на странице развернётся стартовый шаблон. Это похоже на то, когда разворачиваем стартовый код html через ! и tab, только более подробный и с bootstrap.

Дргие плагины

  • AutoFileName - дополняет код при написании путей до файлов в вёрстке
  • Emmet - ускоряет написание HTML и CSS кода до скорости света.
  • terminus - плагин встроенного терминала Sublime Text.
  • W3CValidators - мощный набор валидаторов для HTML разметки, CSS, SVG и т.д
  • Gist - подключает в Sublime Text возможность использовать сервис сниппетов кода GitHub Gist
  • Sass - плагин для подсветки Sass синтаксиса в Sass и Scss файлах. Раскрывает Emmet в Sass файлах.
  • SublimeCodeIntel - автодополнение для javascript

  • Одно из мегаудобных свойств в Sublime Text является возможность вставлять текст, и автоматически выравнивать его по существующему. Да, да, вы не ослышались. Редактор может выровнять вставленный кусок кода по существующему шаблону. Для обычной вставки используется сочетание клавиш Ctrl+V, а для вставки и выравнивания Ctrl+Shift+V. Кстати эти команды можно найти в меню Edit.Если вы много работаете с кодом, то нажимать Ctrl+Shift+V приходится значительно чаще чем Ctrl+V. Так может стоит переназначить их функционал? Давайте! Зайдите в preferences → Key Bindings – User. Вставьте код: { "keys": ["ctrl+v"], "command": "paste_and_indent" },
    { "keys": ["ctrl+shift+v"], "command": "paste" }
  • Как вставить и отформатировать текст мы уже посмотрели. А если код существует, и мы хотим в Sublime Text автоматически выровнять его? Все просто, выделяем неровный кусок и Edit → Line → Reindent и у нас всё выровняется. Однако сочетания клавиш для данной функции нет. Создаем. Откройте preferences → Key Bindings - User . Здесь добавьте: { "keys": ["ctrl+shift+r"], "command": "reindent" }
  • Чтобы заработало оборачивание слов в теги сделаем так: Откроем Preferences->Key Binding и справа в пользовательских настройка клавишь добавим строку: { "keys": ["ctrl+shift+g"], "command": "emmet_wrap_with_abbreviation" }, внутрь массива.
