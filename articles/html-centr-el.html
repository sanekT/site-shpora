Центрирование элемента внутри блока-родителя

Возьмём блок и внутрь поместим картинку, чтобы картинку центрировать внутри этого блока есть 4 способа:

  • Первый: 
                    .block-parent{
                    width: 300px;
                    height: 300px;
                    
                    border: 1px solid #ccc;
                    border-radius: 50%;
                    
                    position: relative;
                }
                .img-child{
                    max-width: 50%;//эти размеры для того, чтобы картинка не вылезала за родителя, если она вдруг больше размером, чем родитель
                    max-height: 50%;
                   
                    position: absolute;
                    top: 0;
                    left: 0;
                    right: 0;
                    bottom: 0;
                    z-index: 1;
                    margin: auto;
                }
  • Второй: 
                    .block-parent{
                    width: 300px;
                    height: 300px;
                    
                    border: 1px solid #ccc;
                    border-radius: 50%;
                    
                    position: relative;
                }
                .img-child{
                    max-width: 50%;//эти размеры для того, чтобы картинка не вылезала за родителя, если она вдруг больше размером, чем родитель
                    max-height: 50%;
                   
                    position: absolute;
                    top: 50%;
                    left: 50%;
                    z-index: 1;
                    
                    transform: translate(-50%, -50%)
                    //также можно и отрицательным margin отодвинуть на половину элемента
                }
  • Третий - с помощью флексов, более современный способ: 
                    .block-parent{
                    display: flex;
                    justify-content: center;
                    align-items: center;
                    width: 300px;
                    height: 300px;
                    
                    border: 1px solid #ccc;
                    border-radius: 50%;
                }
                .img-child{
                    max-width: 50%;//эти размеры для того, чтобы картинка не вылезала за родителя, если она вдруг больше размером, чем родитель
                    max-height: 50%;
                }
  • Четвёртый: 
                    .block-parent{
                    display: flex;
                    width: 300px;
                    height: 300px;
                    
                    border: 1px solid #ccc;
                    border-radius: 50%;
                }
                .img-child{
                    max-width: 50%;//эти размеры для того, чтобы картинка не вылезала за родителя, если она вдруг больше размером, чем родитель
                    max-height: 50%;
                    
                    margin: auto;
                }

    • Другие способы вырявнять вертикально элементы

    Вертикально отцентрировать одну строку в эелементе с известной высотой можно указав эту высоту в свойстве line-height.
